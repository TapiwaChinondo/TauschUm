# TauschUm

TauschUm is planned full-stack trading marketplace that my supervisor has been thinking of makaing for a while. Its a platform where users trade goods directly with one another instead of buying and selling with money. I thought making a prototype would be a nice project to have fun with, without the compelxities of actuallly deploying this.

Also an opportunity to work on frontend which I rarely do.

## Planned Features

- User accounts 
- Item listings
- Image uploads
- Search
- Item trade requests
- Trade negotiation system (Chat possibly included)
- Secure authentication (data encoding)
- Responsive frontend (scary)

## Tech Stack (Planned)

- Python
- FastAPI or Flask  (for a server)
- SQL Database
- Git

Look into React and or other technologies for the frontend
